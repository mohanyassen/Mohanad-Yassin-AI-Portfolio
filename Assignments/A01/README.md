# A01 — A Comparative Analysis of Machine Learning and Deep Learning Tools and Frameworks

## Overview

This assignment explores the differences between machine learning and deep learning tools and frameworks commonly used in artificial intelligence development.

The project focuses on understanding the strengths, weaknesses, and use cases of different AI technologies and development environments.
