# A03 — Deep Learning for an 11 Year Old

## Overview

This assignment focuses on explaining deep learning concepts in a simple and beginner-friendly way that could be understood by an 11-year-old audience.

The goal was to simplify complex artificial intelligence concepts using clear language, simple examples, and relatable explanations.
