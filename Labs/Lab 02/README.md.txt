# L02 — Neural Networks from Scratch to Frameworks

## Overview

This lab explores the fundamentals of neural networks and how they are implemented both manually and through modern deep learning frameworks.

The lab demonstrates the transition from understanding neural network logic from scratch to using high-level AI libraries and frameworks for model development.

---

# Objectives

* Understand the structure of neural networks
* Learn how neurons, weights, and activation functions work
* Explore forward propagation concepts
* Compare manual implementations with framework-based implementations
* Gain familiarity with deep learning workflows

---

# Topics Covered

* Artificial Neural Networks (ANNs)
* Neurons and Layers
* Activation Functions
* Forward Propagation
* Deep Learning Frameworks
* Neural Network Architecture

---

# Technologies Used

* Python
* Jupyter Notebook
* Deep Learning Libraries

---

# Key Learning Outcomes

Through this lab, I learned:

* How neural networks process information
* The role of weights and activations in prediction
* How deep learning frameworks simplify AI model creation
* The importance of neural network architecture design
