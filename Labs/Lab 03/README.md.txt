# L03 — CNN 2026

## Overview

This lab explores Convolutional Neural Networks (CNNs) and their applications in image recognition and computer vision tasks.

The lab focuses on understanding how CNN architectures process image data and extract visual features for classification and prediction.

---

# Objectives

* Understand the fundamentals of Convolutional Neural Networks
* Learn how CNNs process image data
* Explore feature extraction and convolution operations
* Practice building and testing CNN-based models
* Analyze image classification workflows
